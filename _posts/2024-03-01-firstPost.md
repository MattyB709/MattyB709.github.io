## This Past Week
This past week I wrote a data compression algorithm, learned how to access my computer from anywhere on the internet, interviewed the next year of kids for GT Mentorship, and played poker. 

**Data Compression Algorithm**
This was the task Dr. Mamouras (my mentor at Rice) gave me last Friday. Basically, given a series of numbers that take up, let's say, 10 bytes of data, I had to find a way to store that data using less bytes in a way that would let someone else reconstruct the original message without losing any information. I did it in the simplest way possible based on the [research paper](https://arxiv.org/abs/1808.02515) he gave me, but it was still hard as hell. Working with 0's and 1's is not intuitive. When I finished, he called me out on a few issues I had, but still told me he was impressed with how much I was able to learn in the past week, because I had never worked with bits before.

**Connecting to my computer**
This was a game changer. I learned to dual boot my computer so that I could use Windows and Linux (specifically Ubuntu), and then configured my router to allow certain connections to go my computer, which means now I can access Linux on my computer from anywhere. This also sucked to do, I had to learn a ton about the internet (IP addresses, port forwarding, Secure Shell, how my router works), but now that I'm done, me and Uddhav can run our code for SeizureSense on my computer even while we're at school, which will be a huge boost to our productivity. Lowkey I need to add more security, because I may have opened up my computer to be hacked by anyone that guesses my password. Oh well, should be fine.
